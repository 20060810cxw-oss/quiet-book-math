# 安静书数学乐园发布说明

这个文件夹是网站发布版。

## 文件结构

- `index.html`：网站首页，直接部署这个文件。
- `看图题素材/`：题目图片素材，必须和 `index.html` 放在同一级目录下。

## 推荐发布方式

### 方式一：GitHub + Netlify 自动更新

这是后续长期维护推荐方式。

1. 打开 https://github.com/new
2. 仓库名可以填：`quiet-book-math`
3. 选择 `Public`。
4. 创建仓库后，把 `网站发布版` 文件夹里的所有内容上传到仓库根目录。
5. 回到 Netlify，选择从 GitHub 导入项目。
6. Build command 留空。
7. Publish directory 填 `.`。
8. 部署完成后，以后只要更新 GitHub 仓库，Netlify 会自动更新网站。

如果你把 GitHub 仓库创建好并告诉我仓库名，我后面可以直接帮你改仓库里的网页文件。

### 方式二：Netlify Drop

1. 打开 https://app.netlify.com/drop
2. 把整个 `网站发布版` 文件夹拖进去。
3. 等待上传完成，Netlify 会给一个网址。
4. iPad 用 Safari 打开这个网址即可。

### 方式三：GitHub Pages

1. 新建一个 GitHub 仓库。
2. 上传 `网站发布版` 文件夹里的所有内容。
3. 在仓库设置里开启 Pages。
4. iPad 打开 GitHub Pages 生成的网址。

## 注意

不要只上传 `index.html`，否则卷子题图片会丢失。
